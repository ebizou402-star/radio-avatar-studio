# 品質ゲート

制作部隊から修正部隊への引き渡し後、次を実行する。

```bash
node tools/quality_gate.mjs
```

成功時は、構文、配布ビルド、主要機能、セキュリティ、素材参照、CSPハッシュ、ZIPの検査結果が表示される。

## 手動確認

自動検査の後、HTTPS版で次を確認する。

1. マイクを許可すると左右の人物と口が音声に反応する。
2. 録音を開始・停止すると録音ファイルが作成される。
3. ホストと携帯ゲストが招待・回答コードで接続できる。
4. 録音完了または切断後、両方のコード欄が空になる。
5. デスクトップと携帯で操作ボタンや人物が重ならない。

ブラウザや端末の制限で確認できない項目は、最終報告で未確認として扱う。
